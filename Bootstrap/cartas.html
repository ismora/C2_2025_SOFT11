Cartas (Cards) con Bootstrap

Las cartas son componentes versátiles ideales para presentar información de forma ordenada y visualmente atractiva.

Código de ejemplo

<div class="card" style="width: 18rem;"> <img src="imagen.jpg" class="card-img-top" alt="..."> <div class="card-body"> <h5 class="card-title">Título</h5> <p class="card-text">Contenido descriptivo.</p> <a href="#" class="btn btn-primary">Leer más</a> </div> </div>

Modificaciones posibles:

  • Color: Usar bg-success, bg-warning, etc.
  • Contenido: Incluir listas, botones extra o íconos.
  • Diseño: Combinar varias cartas en columnas.
