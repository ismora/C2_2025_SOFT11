Botones en Bootstrap

Bootstrap ofrece una variedad de botones listos para usar con estilos, tamaños e íconos.

Código de ejemplo

<button type="button" class="btn btn-primary">Guardar</button>
<button type="button" class="btn btn-outline-danger">Eliminar</button>
<button type="button" class="btn btn-warning" disabled>Procesando...</button>

Modificaciones posibles:

  • Colores: btn-success, btn-danger, etc.
  • Tamaño: btn-sm, btn-lg.
  • Deshabilitar: disabled para inhabilitar botón.
  • Íconos: Combinar con Bootstrap Icons (si están cargados).
