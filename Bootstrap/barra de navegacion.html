Barra de navegación (nav)

Es un elemento que se utiliza para ofrecer enlaces a otras secciones o páginas de un sitio web. Generalmente se encuentra en la parte superior del sitio y mejora la experiencia de navegación para los usuarios.

Código de ejemplo para crear una barra de navegación:


<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container">
        <a class="navbar-brand" href="#">Mi Sitio</a>
    </div>
</nav>

Resultado:

Barra de navegación

Posibles modificaciones al código de ejemplo:

  • Color de fondo: Cambiar bg-dark por bg-primary, bg-light, bg-success, entre otros.
  • Estilo del texto: Cambiar navbar-dark por navbar-light si se utiliza un fondo claro.
  • Texto del sitio: Modificar Barra de navegación por el nombre de la página.
  • Agregar enlaces: Incluir elementos de navegación dentro del contenedor.

Ejemplo con enlaces y color diferente:


<nav class="navbar navbar-expand-lg navbar-light bg-primary">
    <div class="container">
        <a class="navbar-brand" href="#">Mi Página</a>
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link text-white" href="#">Inicio</a>
            </li>
            <li class="nav-item">
                <a class="nav-link text-white" href="#">Contacto</a>
            </li>
        </ul>
    </div>
</nav>

Resultado de las modificaciones:

Mi Página
  • Inicio
  • Contacto
