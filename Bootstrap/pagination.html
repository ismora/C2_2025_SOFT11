Componentes de Bootstrap: Pagination (Paginación)

Funcionalidad: La paginación permite dividir contenido en varias páginas con enlaces para navegar entre ellas.

Características:

  • Fácil de implementar con clases como .pagination.
  • Admite diferentes tamaños y estilos activos/deshabilitados.
  • Puede personalizarse con íconos o texto.
