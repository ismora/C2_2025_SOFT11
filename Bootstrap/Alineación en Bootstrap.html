Alineación en Bootstrap v.1

1. Contenedor Flex (d-flex)

Primero necesitamos establecer el contenedor como flexbox con d-flex.

<div class="d-flex"> <!-- Elementos a alinear --> </div>

Ejemplo:

Item 1
Item 2
Item 3

2. Alineación Horizontal (justify-content)

Controla la alineación a lo largo del eje principal (horizontal por defecto).

justify-content-start (por defecto)

<div class="d-flex justify-content-start">
Start
Start

justify-content-center

<div class="d-flex justify-content-center">
Center
Center

justify-content-end

<div class="d-flex justify-content-end">
End
End

justify-content-between

<div class="d-flex justify-content-between">
Between
Between

justify-content-around

<div class="d-flex justify-content-around">
Around
Around

3. Alineación Vertical (align-items)

Controla la alineación a lo largo del eje transversal (vertical por defecto).

align-items-start

<div class="d-flex align-items-start" style="height: 150px;">
Start
Start

align-items-center

<div class="d-flex align-items-center" style="height: 150px;">
Center
Center

align-items-end

<div class="d-flex align-items-end" style="height: 150px;">
End
End

align-items-stretch (por defecto)

<div class="d-flex align-items-stretch" style="height: 150px;">
Stretch
Stretch
