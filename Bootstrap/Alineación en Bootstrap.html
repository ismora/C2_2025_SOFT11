1. Contenedor Flex (d-flex)

Primero necesitamos establecer el contenedor como flexbox con d-flex.

4. Alineación de Elementos Individuales (align-self)

Puedes alinear elementos individuales dentro de un contenedor flex.

<div class="d-flex" style="height: 150px;"> <div class="align-self-start">Arriba</div> <div class="align-self-center">Centro</div> <div class="align-self-end">Abajo</div> </div>

7. Alineación Responsive

Bootstrap permite controlar la alineación en diferentes breakpoints usando sufijos como -sm, -md, -lg, -xl, -xxl.

Ejemplo: Cambiar alineación en pantallas medianas

8. Centrado Perfecto

Para centrar un elemento tanto horizontal como verticalmente en su contenedor:

<div class="d-flex justify-content-center align-items-center" style="height: 300px;"> <div class="text-center"> <h3>Contenido centrado</h3> <p>Tanto horizontal como verticalmente</p> </div> </div>

9. Espaciado y Ordenamiento

Complementando la alineación, Bootstrap ofrece utilidades para espaciado y ordenamiento.

Margenes Automáticos (mx-auto, my-auto)

<div class="d-flex" style="height: 150px;"> <div class="mx-auto my-auto demo-item">Centrado con márgenes</div> </div>

Resumen de Alineación en Bootstrap

  • d-flex - Crea un contenedor flex
  • justify-content-* - Alineación horizontal
  • align-items-* - Alineación vertical
  • align-self-* - Alineación individual
  • flex-direction-* - Dirección de los elementos
  • flex-wrap-* - Control de múltiples líneas
  • Clases responsive con breakpoints (-sm, -md, etc.)
  • mx-auto, my-auto - Centrado con márgenes
  • order-* - Control del orden visual
  • text-* - Alineación de texto
