Componentes de Bootstrap: Badges (Insignias)

Funcionalidad: Las insignias se utilizan para mostrar contadores, etiquetas o información destacada, comúnmente junto a otros elementos como botones o encabezados.

Características:

  • Soportan diferentes colores (utilizando clases como .bg-primary, .bg-success, etc.).
  • Se pueden usar con botones o textos.
  • Diseñadas para ser compactas y visualmente llamativas.
