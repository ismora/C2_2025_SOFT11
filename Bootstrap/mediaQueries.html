Media queries

Condicionales en CSS que permite aplicar estilos específicos según las características del dispositivo o del entorno de visualización.

Estilo base (dispositivos móviles)

.element {
  propiedad: valor-xs;
}

@media (min-width: 576px) {
  .element {
  propiedad: valor-sm;
  }
}

@media (min-width: 768px) {
  .element {
  propiedad: valor-md;
  }
}

Tabletas (de 576px a 991.98px)

@media (min-width: 576px) and (max-width: 991.98px) {
 .card {
   flex: 1 1 calc(50% - 20px);
 }

 header {
   padding: 1.5rem;
 } }

Modo oscuro según preferencia del sistema

@media (prefers-color-scheme: dark) {
  body {
   background-color: #121212;
   color: #f0f0f0;
  }

  .card,
  footer,
  nav ul {
   background-color: #1e1e1e;
   color: #f0f0f0;
  }

  .card h2 {
   color: #64b5f6;
   border-bottom-color: #333;
  }

  nav ul li a {
   color: #64b5f6;
  }

  nav ul li a:hover {
   background-color: #64b5f6;
   color: #121212;
  } }
